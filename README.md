## API & Logic

ZenWallet provides a comprehensive API surface for both frontend interactions and external integrations.

- **`/api/core/*`**: Direct interaction with ZenWallet core features
- **`/api/data/*`**: Market data and analytics
- **`/api/user/*`**: User profiles and settings
